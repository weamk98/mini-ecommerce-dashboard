# Mini E-Commerce Dashboard

A small PHP + MySQL project to manage products.

## Features
- Add products
- Delete products
- Stored in MySQL
- Simple clean UI
- Uses prepared statements (basic security)

## Tech
- PHP
- MySQL
- HTML/CSS
- Git

## Setup
1. Create DB `mini_shop` and table `products`
2. Put project into your web server folder (e.g., XAMPP `htdocs`)
3. Open `http://localhost/mini-ecommerce-dashboard/`
